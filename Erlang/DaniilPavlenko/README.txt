Чтобы запустить, нужно:
1) скачать и установить Erlang https://www.erlang.org/downloads, опционально добавить папку с бинарниками в PATH
2) запустить Erlang (erl в коносли, если путь есть в PATH) в папке с файлами .erl
3) в консоли Эрланга ввести (чтобы скомпилировать модули): 
$ c(webserver_daniil).
$ c(webserver_daniil_parallel_rows).
4) запустить сервер с нераспаралеленным умножением матриц:
$ webserver_daniil:start().
В скобочки можно написать цифру порта, по умолчанию 27015
Чтобы выключить сервер (и освободить сокет) нужно выйти из консоли эрланга
5) запустить сервер с распаралеленным умножением матриц:
$ webserver_daniil_parallel_rows:start().

Версия с распаралеленным уминожением почему-то работает у меня нестабильно (то быстрее обычной, то медленнее) и крашит мою IDE. Разбираться времени не осталось.
Обе версии запускают сервер который может принимать и обрабатывать запросы параллельно (одновременно), разница только в умножении матриц.
